# Organizador de Compras

## Descripcion

Aplicacion web para organizar las compras del supermercado. Permite agregar productos, marcarlos como comprados y eliminarlos.

## Funcionalidades

- Agregar items con nombre y cantidad
- Marcar items como listos (se tachan)
- Quitar items de la lista
- Ver cuantos items hay en total, listos y pendientes
- Los datos se guardan automaticamente en el navegador

## Tecnologias

- HTML5
- CSS3 (con Flexbox)
- JavaScript

## Uso

1. Abrir index.html en el navegador
2. Escribir el nombre del producto
3. Escribir la cantidad
4. Presionar Agregar
5. Click en el item para marcarlo como listo
6. Click en Quitar para eliminarlo
